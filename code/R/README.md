# Readme 

Scripts for the pilot are marked with "00" at the start of the file name.

Scripts that are specific to paper 1 are either marked with "A" at the start of the filename, or with "_ACQ" at the end of the filename.

Scripts that are specific to paper 2 are either marked with "R" or "Rfam" at the start of the filename, or with "_REC" at the end of the filename.

The file AR00_prepare_data_exp.R is used for both papers.


Please note that the speaker numbering in this GitHub repository differs from the speaker numbering in the papers:
Numbering GitHub          | Numbering papers                         
------------------------- | --------------------------------
Speaker 1                 | Speaker 1            
Speaker 2                 | Speaker 2           
Speaker 3                 | Speaker 4               
Speaker 4                 | Speaker 3
